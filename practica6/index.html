Servidor Local XAMPP

- Resumen -

Esta práctica de laboratorio tiene como finalidad instalar y configurar XAMPP para crear un servidor local. Con ello, se busca que el estudiante comprenda el funcionamiento básico de un entorno de desarrollo web local, pueda ejecutar archivos PHP y gestionar servicios como Apache y MySQL, facilitando así las pruebas y el desarrollo de aplicaciones web en un entorno controlado.

- Introducción -

En el desarrollo de aplicaciones web, es fundamental contar con un entorno de pruebas que permita ejecutar y verificar el funcionamiento del código antes de ser implementado en un servidor de producción. XAMPP es una herramienta ampliamente utilizada para este propósito, ya que permite instalar y configurar fácilmente un servidor local que incluye Apache, MySQL, PHP y Perl. Esta práctica de laboratorio tiene como objetivo principal la instalación y configuración de XAMPP en un entorno local, así como la comprensión de su estructura y funcionamiento. A través de esta actividad, se busca que el estudiante adquiera habilidades básicas para levantar un servidor local, gestionar servicios web y probar archivos PHP de manera efectiva, sentando así las bases para futuros desarrollos en entornos controlados y seguros.

- Materiales -

  • Laptop
  • Plataforma XAMPP
  • Botón y resistencia de 1Khom
  • Conexión a Internet

- Desarrollo -

Para la instalación de XAMPP, se debe descargar el instalador desde la página oficial de Apache Friends. Una vez descargado, se procede a ejecutar el instalador y seguir las instrucciones en pantalla. Durante la instalación, se seleccionan los componentes que se desean instalar (Apache, MySQL, PHP, etc.) y se elige la carpeta de destino para la instalación.

Una vez instalado, se inicia el panel de control de XAMPP, donde se pueden gestionar los servicios de Apache y MySQL. Se recomienda iniciar ambos servicios para poder ejecutar aplicaciones web que requieran una base de datos.

Para probar el funcionamiento de XAMPP, se puede crear un archivo PHP simple en la carpeta "htdocs" de la instalación de XAMPP. Este archivo puede contener un código básico que muestre información del servidor, como la versión de PHP instalada y la configuración del entorno.

Una vez creado el archivo, se abre un navegador web y se ingresa la dirección "http://localhost/nombre_del_archivo.php" para verificar que el servidor local esté funcionando correctamente. Si todo está configurado adecuadamente, se debería ver la salida del código PHP en el navegador.

Además, se puede utilizar phpMyAdmin, una herramienta web incluida en XAMPP, para gestionar bases de datos MySQL. Para acceder a phpMyAdmin, se ingresa la dirección "http://localhost/phpmyadmin" en el navegador. Desde allí, se pueden crear bases de datos, tablas y realizar consultas SQL.

En resumen, la instalación y configuración de XAMPP permite crear un entorno de desarrollo web local que facilita la creación y prueba de aplicaciones PHP y la gestión de bases de datos MySQL. Esta práctica proporciona una base sólida para el desarrollo de proyectos web en un entorno controlado.
